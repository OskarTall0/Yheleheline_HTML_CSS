Mis on HTML?

HTML on veebilehtede põhiline keel, mida kasutatakse selleks, et määrata lehe sisu ja struktuur. Selle abil öeldakse arvutile, millised osad on pealkirjad, lõigud, lingid, pildid või muud elemendid. HTML ei tegele kujundamise ega värvidega, vaid ainult selle määramisega, mis tüüpi sisu kuskil asub. See on justkui vundament, mille peale ehitatakse ülejäänud veebileht. HTML-i mõistmine on vajalik, kui soovid luua või muuta veebilehti.

HTML tähendus

HTML tähendab HyperText Markup Language, eesti keeles hüperteksti märgistuskeel. See on keel, mida kasutatakse veebilehtede loomiseks ja struktuuri määratlemiseks. HTML ei ole programmeerimiskeel, vaid märgistuskeel - see tähendab, et see kirjeldab, milline sisu on lehel ja kuidas see on üles ehitatud (näiteks pealkirjad, lõigud, pildid, lingid jms). HTML on veebilehe alus, millele lisatakse kujundus CSS-iga ja funktsionaalsus JavaScriptiga.
